void open_fileRW ( fstream & file,

Opens a file named filename for reading and writing using the passed in fstream parameter file.

The passed in fstream instance, file, is closed if open at the time of being passed as an actual parameter.

If a file with the name filename already, exists, open_fileRW simply opens the file for read and write, and does not overwrite the existing file. If not, opens the file for write and creates a new file.
